# Проект Moneta (Курсовая робата).

## Цель проекта:

### ***Разрабат приложение для анализа транзакций, которые находятся в Excel-файле. Приложение будет генерировать JSON-данные для веб-страниц, формировать Excel-отчеты, а также предоставлять другие сервисы.***

**Инструкция по установке:**

Для работы проекта требуется [Poetry](https://python-poetry.org/).

    1. Клонируйте репозиторий:
      git clone https://github.com/burmspb-code/Coursework.git
        cd Coursework
    2. Установите зависимости:
        poetry install
    3. Запустите проект:
        poetry run python src/main.py

## 🧪 Тестирование:

Для проверки корректности работы модулей Moneta используется фреймворк pytest.
Тесты покрывают основные функции загрузки, фильтрации и обработки данных.

Проверяемые модули: views, services, reports, main.

После запуска  pytest --cov=src --cov-report=html подробный отчет доступен в папке htmlcov/index.html

**Используемые модули:**

    - views,
    - services,
    - reports,
    - main.

### **Views**

Модуль содержит функции обработки данных:

    load_xlsx - функция загрузки xlsx данных из указнанного файла;
    get_ExchangeRate - получение курса валюты через API сайта ExchangeRate;
    get_stock_price - получение стоимости акции на текущую дату;
    get_operations - возвращает датафрейм из исходного за указанный период;
    get_summary_stats - формирует данные в формате JSON.

### **Services**

Модуль содержит функции обработки данных:

    analyze_cashback_profit - aнализ категории кэшбэк.

### **Reports**

Модуль содержит функции обработки данных:

    report_to_excel - декоратор для записи результата DataFrame в Excel файл;
    spending_by_category - Возвращает траты по заданной категории за последние три месяца от переданной даты.

### **Main**

Модуль служит для проверки и связи всех функциональностей проекта.
